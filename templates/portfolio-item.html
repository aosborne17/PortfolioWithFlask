Creating an EC2 Instance Using AWS

Project Descripton here

During this project I was effectively able to create an EC2 instance that would be hosted inside a VPC an in a public subnet.

This public subnet is secured through a Network ACL which ensures only specific traffic can enter. In addition the security groups within this Instance only allowed users to access at port 80, as this is where HTTP is present

This was then coupled with further stages where the server was managed trough Ansible and provisioned with all the necessary dependencies such as Docker and DockerCompose
